Formulario trigonométrico

En un triángulo rectángulo, todo se mide respecto del ángulo agudo α: el cateto frente a él es el opuesto, el que lo toca es el adyacente y el lado mayor es la hipotenusa.

Seno · SOH
sen α = opuesto / hipotenusa
Coseno · CAH
cos α = adyacente / hipotenusa
Tangente · TOA
tan α = opuesto / adyacente
Identidad fundamental
sen²α + cos²α = 1
tan α = sen α / cos α
Ángulos notables · seno
sen 30° = ½ · sen 45° = √2/2 · sen 60° = √3/2
Ángulos notables · coseno
cos 30° = √3/2 · cos 45° = √2/2 · cos 60° = ½
Ángulos notables · tangente
tan 30° = √3/3 · tan 45° = 1 · tan 60° = √3
Teorema de Pitágoras
a2 + b2 = c2
Útil para hallar un lado que falta antes de calcular una razón.

Introducción · Función y ecuación cuadrática

15 min

La función cuadrática modela todo lo que sube y baja describiendo una curva: la trayectoria de una pelota, la forma de un arco, las ganancias que crecen y luego caen. Su gráfico es una parábola.

Función cuadrática
f(x) = ax2 + bx + c  (a ≠ 0)
Su gráfico es una parábola. Corta el eje Y en (0, c).
Concavidad
a > 0 → abre hacia arriba (mínimo)
a < 0 → abre hacia abajo (máximo)
Vértice y eje de simetría
xv = −b / (2a)
El vértice es el punto más alto o más bajo; el eje de simetría es la recta x = −b/(2a).
Ecuación cuadrática
ax2 + bx + c = 0
Resolverla es buscar los ceros (raíces) de la función, donde la parábola corta el eje X.
Fórmula general
x = ( −b ± √(b2 − 4ac) ) / 2a
Discriminante Δ = b² − 4ac
Δ > 0 → 2 soluciones
Δ = 0 → 1 solución · Δ < 0 → ninguna real
