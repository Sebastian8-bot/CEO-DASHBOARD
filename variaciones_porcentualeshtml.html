1. Definición de Variación Porcentual

La variación porcentual es una medida que indica cómo ha cambiado una cantidad en relación con su valor inicial, expresada como un porcentaje.

V% = (Valor Final - Valor Inicial) / Valor Inicial × 100 O de forma más compacta: V% = (Vf - Vi) / Vi × 100

📌 Ejemplo Básico:

Un producto cuesta $100 y su precio sube a $120.

V% = (120 - 100) / 100 × 100 = 20 / 100 × 100 = 20%

El precio aumentó en un 20%.

2. Interpretación del Signo

Tipo de Variación Signo Significado Ejemplo
Aumento Positivo (+) El valor final es mayor +15% (creció)
Disminución Negativo (-) El valor final es menor -10% (decreció)
Sin cambio Cero (0) El valor permanece igual 0% (estable)

3. Cálculo del Valor Final

Si conocemos la variación porcentual y queremos encontrar el valor final:

Vf = Vi × (1 + r%) Donde r% es la tasa de variación en forma decimal. Ejemplo: si r% = 20%, entonces usamos 1 + 0.20 = 1.20

📌 Ejemplo de Cálculo hacia Adelante:

Un artículo cuesta $500 y aumenta un 15%.

Método 1: Vf = 500 + (500 × 0.15) = 500 + 75 = $575

Método 2: Vf = 500 × 1.15 = $575

El método 2 es más práctico.

4. Cálculo del Valor Inicial

Si conocemos el valor final y la variación, podemos encontrar el valor inicial:

Vi = Vf / (1 + r%)

📌 Ejemplo de Cálculo hacia Atrás:

Después de una disminución de 20%, un producto cuesta $800.

Si disminuye 20%, queda el 80% del original (o 0.80).

Vi = 800 / 0.80 = $1.000

Comprobación: 1.000 × 0.80 = 800 ✓

5. Variaciones Compuestas (Múltiples Cambios)

Cuando hay varios cambios porcentuales sucesivos, estos se multiplican, no se suman.

Vf = Vi × (1 + r₁%) × (1 + r₂%) × (1 + r₃%) ...

⚠️ Error Común:

Si algo sube 10% y luego sube 10%, el aumento total NO es 20%.

Vf = Vi × 1.10 × 1.10 = Vi × 1.21

El aumento total es 21%, no 20%.

6. El "Efecto Cascada": Aumentos y Descuentos

Un aumento seguido de un descuento del mismo porcentaje NO resultan en el valor original.

📌 Ejemplo:

Un artículo cuesta $100.

▸ Sube 20%: 100 × 1.20 = $120

▸ Baja 20%: 120 × 0.80 = $96

Resultado final: $96 (NO $100)

Variación total: (96 - 100) / 100 × 100 = -4%

El descuento se aplica sobre una base mayor, por eso "cuesta más".

7. Aplicación a Interés Compuesto

La variación porcentual compuesta es la base para calcular intereses en inversiones:

Vf = Vi × (1 + r%)ⁿ Donde n es el número de períodos

📌 Ejemplo:

Una inversión de $10.000 crece 5% anualmente durante 3 años.

Vf = 10.000 × (1.05)³ = 10.000 × 1.157625 = $11.576,25

Variación total: (11.576,25 - 10.000) / 10.000 × 100 = 15.76%

8. Resumen de Fórmulas

Situación Fórmula Cuándo Usarla
Encontrar variación % V% = (Vf - Vi) / Vi × 100 Conoces Vi y Vf
Encontrar Vf Vf = Vi × (1 + r%) Conoces Vi y r%
Encontrar Vi Vi = Vf / (1 + r%) Conoces Vf y r%
Múltiples cambios Vf = Vi × (1+r₁) × (1+r₂) ... Hay varios cambios
Cambios periódicos Vf = Vi × (1 + r%)ⁿ Cambios regulares n veces
