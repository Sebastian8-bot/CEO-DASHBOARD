Parte 1

¿Qué es un logaritmo?

Antes de cualquier ejercicio, hay que entender qué significa la expresión.

La idea fundamental

Un logaritmo responde a la pregunta: "¿a qué potencia tengo que elevar la base para obtener ese número?"

logb(x) = y  ⟺  by = x
b
BASE
b > 0, b ≠ 1
x
ARGUMENTO
x > 0 siempre
y
RESULTADO
el exponente
Ejemplo concreto: log₂(8) = 3 porque 2³ = 8. La base es 2, el argumento es 8, el resultado es 3.
→
Conversión básica
Convierte entre forma logarítmica y exponencial:
log₃(81) = ?
  • 1
    Identifica los elementos
    Base = 3  |  Argumento = 81  |  Pregunta: ¿a qué potencia elevo 3 para obtener 81?
  • 2
    Descompón el argumento en potencia de la base
    81 = 3 × 3 × 3 × 3 = 3⁴
    81 = 3⁴
  • 3
    Aplica la definición
    Si 3⁴ = 81, entonces log₃(81) = 4
    log₃(81) = 4  porque  3⁴ = 81 ✓
✓
Respuesta
log₃(81) = 4
Parte 2

Las 8 propiedades esenciales

Memoriza estas — son las herramientas con las que se resuelve todo.

① Propiedad del Producto
log_b(M · N) = log_b(M) + log_b(N)
Multiplicación adentro → suma afuera
Ej: log(2·5) = log2 + log5
② Propiedad del Cociente
log_b(M/N) = log_b(M) − log_b(N)
División adentro → resta afuera
Ej: log(10/2) = log10 − log2
③ Propiedad de la Potencia
log_b(Mᵖ) = p · log_b(M)
El exponente baja y multiplica
Ej: log(8) = log(2³) = 3·log2
④ log de la propia base
log_b(b) = 1
Siempre da 1. La base elevada a 1 es ella misma.
Ej: log₅(5) = 1  |  log(10) = 1
⑤ log de 1
log_b(1) = 0
Siempre da 0. Cualquier base elevada a 0 es 1.
Ej: log₃(1) = 0  |  log(1) = 0
⑥ Base recíproca
log_(1/b)(x) = −log_b(x)
Invertir la base cambia el signo del resultado
Ej: log_(1/2)(8) = −log₂(8) = −3
⑦ Cambio de base
log_b(x) = log(x) / log(b)
Sirve para calcular con calculadora o comparar bases
Ej: log₄(8) = log8/log4 = 3log2/2log2 = 3/2
⑧ Logaritmo compuesto
log_a(b) · log_b(c) = log_a(c)
Se "cancela" la base intermedia. Muy útil en difíciles.
Ej: log₂(3)·log₃(8) = log₂(8) = 3
🧠 Truco para recordar producto y cociente
Piensa en las potencias: cuando multiplicas potencias de la misma base, sumas los exponentes (aᵐ·aⁿ = aᵐ⁺ⁿ). Cuando divides, restas. Los logaritmos hacen exactamente lo mismo — porque son exponentes.
Parte 3 · 🟢 Nivel Fácil

Ejercicios básicos

Aquí practicas la definición y las propiedades fundamentales una a la vez.

F1
Fácil
Calcula: log₂(32)
① Definición
  • 1
    Pregúntate: ¿2 elevado a qué potencia da 32?
    Base = 2. Necesito encontrar y tal que 2ʸ = 32
  • 2
    Descompón 32 como potencia de 2
    32 = 2 × 2 × 2 × 2 × 2 = 2⁵
  • 3
    Aplica la definición
    log₂(32) = log₂(2⁵) = 5
✓
Respuesta
log₂(32) = 5
F2
Fácil
Calcula: log₅(1/25)
③ Potencia① Definición
  • 1
    Escribe 1/25 como potencia de 5
    1/25 = 1/5² = 5⁻²  (recuerda: 1/aⁿ = a⁻ⁿ)
    1/25 = 5⁻²
  • 2
    Aplica la propiedad de la potencia
    log₅(5⁻²) = −2 · log₅(5)
  • 3
    Aplica log_b(b) = 1
    = −2 · 1 = −2
    log₅(5) = 1 porque la base y el argumento son iguales
✓
Respuesta
log₅(1/25) = −2
F3
Fácil
Si log x = a, ¿cuánto vale log √x?
③ Potencia (raíz = exponente 1/2)
  • 1
    Escribe la raíz como exponente
    √x = x^(1/2)
    La raíz cuadrada es lo mismo que elevar a 1/2. Siempre.
  • 2
    Aplica la propiedad de la potencia: el exponente baja
    log(√x) = log(x^(1/2)) = (1/2) · log(x)
  • 3
    Reemplaza log x = a
    = (1/2) · a = a/2
✓
Respuesta
log √x = a/2
F4
Fácil
Calcula: log₅5 + log₂32 / log₃(1/9)
④ log_b(b)=1③ Potencia
  • 1
    Calcula cada logaritmo por separado
    log₅(5) = 1            ← prop. ④: log_b(b) = 1 log₂(32) = log₂(2⁵) = 5   ← prop. ③: baja el exp. log₃(1/9) = log₃(3⁻²) = −2 ← 1/9 = 3⁻²
  • 2
    Sustituye los valores
    (1 + 5) / (−2) = 6 / (−2) = −3
✓
Respuesta
−3
Parte 4 · 🟡 Nivel Medio

Ejercicios intermedios

Aquí se combinan 2 o más propiedades. Hay que ser ordenado paso a paso.

M1
Medio
Simplifica: (log₂16 − log₃(1/27)) / log₆36
③ Potencia④ log_b(b)=1
  • 1
    Calcula el numerador — cada log por separado
    log₂(16) = log₂(2⁴) = 4 log₃(1/27) = log₃(3⁻³) = −3 Numerador = 4 − (−3) = 4 + 3 = 7
    ⚠ Cuidado: restar un negativo es sumar. 4 − (−3) = 4 + 3 = 7
  • 2
    Calcula el denominador
    log₆(36) = log₆(6²) = 2
  • 3
    Divide
    7 / 2 = 7/2
✓
Respuesta
7/2
M2
Medio
Si log x = a y log y = b, expresa: log(³√xy)
③ Potencia (raíz cúbica = exp. 1/3)① Producto
  • 1
    Convierte la raíz cúbica en exponente
    ³√(xy) = (xy)^(1/3)
  • 2
    Aplica la propiedad de la potencia — el (1/3) baja
    log((xy)^(1/3)) = (1/3) · log(xy)
  • 3
    Aplica la propiedad del producto — log(xy) se separa
    = (1/3) · (log x + log y)
  • 4
    Reemplaza log x = a y log y = b
    = (1/3)(a + b) = a/3 + b/3
✓
Respuesta
(a + b)/3
M3
Medio
Desarrolla: log(√(a³b) / c²)
② Cociente① Producto③ Potencia
  • 1
    Separa numerador y denominador con la propiedad del cociente
    log(√(a³b) / c²) = log(√(a³b)) − log(c²)
  • 2
    Convierte la raíz en exponente y baja el (1/2)
    log(√(a³b)) = log((a³b)^(1/2)) = (1/2)·log(a³b)
  • 3
    Separa el producto a³b con la propiedad del producto
    = (1/2)·[log(a³) + log(b)]
  • 4
    Baja el exponente 3 del log(a³)
    = (1/2)·[3·log(a) + log(b)] = (3/2)·log(a) + (1/2)·log(b)
  • 5
    Baja el exponente 2 del log(c²)
    log(c²) = 2·log(c)
  • 6
    Junta todo
    = (3/2)·log(a) + (1/2)·log(b) − 2·log(c)
✓
Respuesta
(3/2)log a + (1/2)log b − 2log c
M4
Medio
Calcula el valor de: log(14/15) + log(12/11) + log(11/8) + log(5/7)
① Producto: suma de logs = log del producto
  • 1
    Suma de logaritmos = logaritmo del producto total
    = log( (14/15) · (12/11) · (11/8) · (5/7) )
  • 2
    Multiplica las fracciones — numeradores y denominadores por separado
    Numerador: 14 × 12 × 11 × 5 = 9.240 Denominador: 15 × 11 × 8 × 7 = 9.240
    Los números se cancelan telescópicamente. Observa: el 11 aparece arriba y abajo.
  • 3
    El resultado es log(1)
    = log(9.240 / 9.240) = log(1) = 0
    log(1) = 0 siempre, en cualquier base — prop. ⑤
✓
Respuesta
0
Parte 5 · 🔴 Nivel Difícil

Ejercicios difíciles

Requieren combinar varias propiedades y a veces hacer conversión de base. Lee cada paso con calma.

D1
Difícil
Calcula: log_{3√3}(1/9 · √3)
③ Potencia① Producto⑦ Cambio de base
🧠 Estrategia
Cuando la base o el argumento tienen raíces o fracciones, primero escribe TODO como potencias de un mismo número (aquí el 3). Después es solo aritmética de exponentes.
  • 1
    Escribe la base como potencia de 3
    3√3 = 3 · 3^(1/2) = 3^(1 + 1/2) = 3^(3/2)
  • 2
    Escribe el argumento como potencia de 3
    (1/9) · √3 = 3⁻² · 3^(1/2) = 3^(−2 + 1/2) = 3^(−3/2)
  • 3
    Aplica la definición: base = 3^(3/2), argumento = 3^(−3/2)
    log_{3^(3/2)} ( 3^(−3/2) ) = y ↓ (3^(3/2))^y = 3^(−3/2) 3^(3y/2) = 3^(−3/2) → 3y/2 = −3/2 → y = −1
✓
Respuesta
−1
D2
Difícil
Si log₂(√17 + 1) = y, calcula log₂(√17 − 1)
① ProductoDiferencia de cuadrados
🧠 La clave oculta
Cuando ves (√17 + 1) y (√17 − 1) juntos, piensa en diferencia de cuadrados: (a+b)(a−b) = a² − b². Multiplicarlos da un número limpio.
  • 1
    Multiplica las dos expresiones — diferencia de cuadrados
    (√17 + 1)(√17 − 1) = (√17)² − 1² = 17 − 1 = 16
  • 2
    Toma logaritmo base 2 en ambos lados
    log₂[(√17+1)(√17−1)] = log₂(16)
  • 3
    Aplica propiedad del producto al lado izquierdo
    log₂(√17+1) + log₂(√17−1) = log₂(16)
  • 4
    Calcula log₂(16) y sustituye log₂(√17+1) = y
    log₂(16) = log₂(2⁴) = 4 y + log₂(√17−1) = 4 log₂(√17−1) = 4 − y
✓
Respuesta
4 − y
D3
Difícil
Demuestra que: log_a(b) · log_b(c) = log_a(c)
⑦ Cambio de base⑧ Log compuesto
  • 1
    Aplica cambio de base a log_a(b)
    log_a(b) = log(b) / log(a)
  • 2
    Aplica cambio de base a log_b(c)
    log_b(c) = log(c) / log(b)
  • 3
    Multiplica ambas expresiones
    log_a(b) · log_b(c) = [log(b)/log(a)] · [log(c)/log(b)] = log(c) / log(a) ← se cancela log(b) = log_a(c) ✓
    El log(b) del numerador y denominador se cancela. Por eso funciona la propiedad ⑧.
✓
Demostración
log_a(b) · log_b(c) = log_a(c) ✓
D4
Difícil
Si x^(log₃5) = 25, calcula x²
③ PotenciaDefinición de log
🧠 Estrategia: aplica log₃ en ambos lados
Cuando la incógnita está en el exponente y el exponente tiene un logaritmo, aplica el logaritmo de la misma base en ambos lados de la ecuación.
  • 1
    Aplica log₃ en ambos lados
    log₃(x^(log₃5)) = log₃(25)
  • 2
    El exponente log₃5 baja (prop. potencia)
    log₃5 · log₃x = log₃(25)
  • 3
    Escribe log₃(25) = log₃(5²) = 2·log₃5
    log₃5 · log₃x = 2·log₃5
  • 4
    Divide ambos lados por log₃5 (que ≠ 0)
    log₃x = 2 → x = 3² = 9
  • 5
    Calcula x²
    x² = 9² = 81
✓
Respuesta
x² = 81

Ejercicios tipo PAES

Formato exacto de prueba — con alternativas y la trampa más común explicada.

P1
PAES
Si log₂√2 = m, log_n(27/8) = −3 y log_{0,2}x = −2, entonces mnx⁻¹ =
③ Potencia① DefiniciónBase 0,2 = 1/5
🧠 Estrategia PAES: calcula cada variable por separado
Cuando hay 3 condiciones separadas, calcula m, n y x individualmente. Luego sustituye. No intentes resolver todo junto.
  • 1
    Calcula m = log₂(√2)
    √2 = 2^(1/2) m = log₂(2^(1/2)) = 1/2
  • 2
    Calcula n de la ecuación log_n(27/8) = −3
    Por definición: n^(−3) = 27/8 1/n³ = 27/8 n³ = 8/27 = (2/3)³ n = 2/3
  • 3
    Calcula x de log_{0,2}(x) = −2
    0,2 = 1/5 = 5⁻¹ Por definición: (1/5)^(−2) = x = 5² = 25 x = 25
  • 4
    Calcula mnx⁻¹ = m · n · (1/x)
    = (1/2) · (2/3) · (1/25) = (1/3) · (1/25) = 1/75 = 0,0133...
    ⚠ x⁻¹ significa 1/x, no −x. Es un error muy frecuente en PAES.
✓
Respuesta
1/75 ≈ 0,013
P2
PAES
Si 4ˣ + 4ˣ + 4ˣ + 4ˣ = 17, el valor de x es:
Suma de iguales① Definición de log
🧠 La trampa clásica
Ver 4ˣ + 4ˣ + 4ˣ + 4ˣ puede confundir. Pero son 4 términos iguales — eso es simplemente 4 · 4ˣ.
  • 1
    Suma los 4 términos iguales
    4 · 4ˣ = 17
  • 2
    Escribe 4 como potencia de 4
    4¹ · 4ˣ = 17 4^(x+1) = 17
  • 3
    Aplica la definición de logaritmo para despejar x+1
    x + 1 = log₄(17) x = log₄(17) − 1
✓
Respuesta
x = log₄17 − 1

PAES
Si log₃(1/3) = x, log_{0,5}y = 3 y log_z(1/2) = −2, entonces xyz =
③ Potencia① Definición
  • 1
    Calcula x = log₃(1/3)
    1/3 = 3⁻¹ log₃(3⁻¹) = −1 x = −1
  • 2
    Calcula y de log_{0,5}(y) = 3
    0,5 = 1/2 (1/2)³ = y y = 1/8
  • 3
    Calcula z de log_z(1/2) = −2
    z^(−2) = 1/2 1/z² = 1/2 z² = 2 z = √2

    xyz = (−1) · (1/8) · (√2) = −√2/8

P4
PAES · Máxima dificultad
Si log_{√2}(16) = (1/x)·log₄(16), entonces x =
⑦ Cambio de base③ Potencia
🧠 Estrategia: calcula cada lado por separado
Cuando ves logaritmos con bases raras como √2, escríbelos en potencias de 2. Todo se simplifica.
  • 1
    Calcula log_{√2}(16)
    √2 = 2^(1/2) y 16 = 2⁴ log_{2^(1/2)}(2⁴) = 4 / (1/2) = 8
    Regla: log_{b^n}(b^m) = m/n. El resultado es 4÷(1/2) = 8.
  • 2
    Calcula log₄(16)
    16 = 4² log₄(4²) = 2
  • 3
    Sustituye en la ecuación original
    8 = (1/x) · 2 8 = 2/x x = 2/8 = 1/4 = 0,25
✓
Respuesta
x = 0,25
⚠ Los 5 errores más frecuentes en la prueba
❌ log(a + b) ≠ log(a) + log(b)
La propiedad del producto solo funciona con multiplicación, NO con suma. log(2 + 3) ≠ log2 + log3.
❌ Olvidar el signo negativo con 1/bⁿ = b⁻ⁿ
log₃(1/9) NO es 2, es −2. La fracción genera exponente negativo.
❌ Confundir x⁻¹ con −x
x⁻¹ = 1/x. Son distintos. En PAES esto aparece como trampa en las alternativas.
❌ Aplicar la propiedad de la potencia mal
log(a²b) NO es 2·log(a)·log(b). Es 2·log(a) + log(b). Primero potencia, luego producto.
❌ Olvidar que el argumento siempre debe ser positivo
log(−5) no existe. Si en una ecuación obtienes x negativo como argumento, esa solución se descarta.
