Nivel: Segundo Medio Duración: 90 minutos Ejercicios: 20
PARTE 01

¿Qué es un logaritmo?

Un logaritmo es simplemente otra forma de escribir una potencia. Cuando te preguntan "¿a qué exponente debo elevar un número para obtener otro?", la respuesta es un logaritmo.

Piénsalo así: si sabemos que 2³ = 8, entonces el logaritmo en base 2 de 8 es 3, porque 3 es el exponente que necesitamos.

Definición
logb(a) = c ⟺ bc = a

Se lee: "logaritmo en base b de a es igual a c". Donde b es la base (b > 0 y b ≠ 1), a es el argumento (a > 0) y c es el exponente.

Ejemplos de conversión

Forma exponencial → logarítmica
2⁵ = 32 → log₂(32) = 5
Forma logarítmica → exponencial
log₃(81) = 4 → 3⁴ = 81
Truco mental La base del logaritmo es la misma base de la potencia. El resultado del logaritmo es el exponente.
PARTE 02

Propiedades fundamentales

Estas son las 8 herramientas que necesitas dominar. Memorízalas porque las usarás en todos los ejercicios.

Propiedad I
Logaritmo de la base
logb(b) = 1
Propiedad II
Logaritmo de 1
logb(1) = 0
Propiedad III
Producto
logb(a·c) = logb(a) + logb(c)
Propiedad IV
Cociente
logb(a/c) = logb(a) − logb(c)
Propiedad V
Potencia
logb(aⁿ) = n · logb(a)
Propiedad VI
Raíz
logb(ⁿ√a) = (1/n)·logb(a)
Propiedad VII
Cambio de base
logb(a) = log(a) / log(b)
Propiedad VIII
Cancelación
logb(bⁿ) = n
Notación importante Cuando ves log(x) sin base escrita, se entiende que la base es 10. Cuando ves ln(x), la base es el número e ≈ 2,718 (logaritmo natural).
PARTE 03

Ejemplos resueltos paso a paso

Antes de empezar los ejercicios, revisa estos ejemplos. Te muestran cómo aplicar las propiedades en distintas situaciones.

Ejemplo 1 · Aplicación directa
Calcular log₂(16)
1
Nos preguntamos: ¿a qué exponente elevamos 2 para obtener 16?
2
Sabemos que 2⁴ = 16, entonces el exponente es 4.
log₂(16) = 4
Ejemplo 2 · Usando propiedad del producto
Calcular log₃(9 · 27)
1
Aplicamos la propiedad III: log₃(9·27) = log₃(9) + log₃(27)
2
Calculamos cada uno: log₃(9) = 2 (porque 3² = 9) y log₃(27) = 3 (porque 3³ = 27).
3
Sumamos: 2 + 3 = 5
log₃(9 · 27) = 5
Ejemplo 3 · Usando propiedad de la potencia
Calcular log₂(8⁵)
1
Aplicamos la propiedad V: el exponente baja multiplicando. log₂(8⁵) = 5 · log₂(8)
2
Calculamos log₂(8) = 3 (porque 2³ = 8).
3
Multiplicamos: 5 · 3 = 15
log₂(8⁵) = 15
Ejemplo 4 · Ecuación logarítmica simple
Resolver log₄(x) = 3
1
Aplicamos la definición: si logb(a) = c, entonces bc = a.
2
Reescribimos: 4³ = x
3
Calculamos: 4³ = 64
x = 64
