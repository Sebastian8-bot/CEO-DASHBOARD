Identidades Logarítmicas — Formulario Completo

DEF
📌 Definición fundamental
logb x = y  ⟺  by = x
El logaritmo de x en base b es el exponente al que hay que elevar b para obtener x. Condiciones: b > 0, b ≠ 1, x > 0.
01
Logaritmo de 1
logb 1 = 0
Para cualquier base b, porque b⁰ = 1.
02
Logaritmo de la base
logb b = 1
Porque b¹ = b.
03
Potencia de la base
blogb x = x
La exponenciación y el logaritmo se cancelan mutuamente.
04
Log de potencia de la base
logb bn = n
Caso especial: log 10n = n (base 10).
05
Logaritmo de un producto
logb(M · N) = logb M + logb N
El log de un producto es la suma de los logaritmos.
06
Logaritmo de un cociente
logb(M / N) = logb M − logb N
El log de un cociente es la diferencia de logaritmos.
07
Logaritmo de una potencia
logb Mn = n · logb M
El exponente «baja» como coeficiente multiplicativo.
08
Logaritmo de una raíz
logb ⁿ√M = (1/n) · logb M
La raíz n-ésima equivale al exponente 1/n.
09
Cambio de base
logb x = logc x / logc b
Permite calcular cualquier logaritmo usando base 10 o base e.
10
Cambio de base — caso especial
logb a · loga c = logb c
Propiedad de cadena de logaritmos (telescópico).
11
Logaritmo decimal (base 10)
log x = log10 x
Cuando no se escribe la base, se asume base 10.
12
Logaritmo natural (base e)
ln x = loge x
e ≈ 2,718… Muy usado en cálculo y modelización.
13
Si b > 1 (base mayor que 1)
logb x > 0 si x > 1  |  logb x < 0 si 0 < x < 1
La función es creciente. Ej: log₁₀ 100 = 2 > 0.
14
Si 0 < b < 1 (base entre 0 y 1)
logb x < 0 si x > 1  |  logb x > 0 si 0 < x < 1
La función es decreciente. Ej: log₀.₅ 2 = −1 < 0.
15
Logaritmo del recíproco
logb(1/x) = −logb x
Equivale a logb x⁻¹ = −logb x.
16
Base recíproca
log1/b x = −logb x
Cambiar la base por su recíproco invierte el signo.
★
Resolución de ecuaciones logarítmicas
Si logb f(x) = logb g(x)  ⟹  f(x) = g(x)   (con f(x) > 0 y g(x) > 0)
Para despejar: si logb x = k, entonces x = bk. Siempre verificar que el argumento sea positivo.

A) 1
B) 2

A) 4
B) 8
C) 16
D) 32
E) 64
Usa: log(M/N) = log M − log N. Entonces log₂(m/n) = 4, así m/n = 2⁴.
11
La expresión log √(x⁵), con x > 0, es igual a:
A) 5 log x
B) (5/2) · log x²
C) (5/2) · log x
D) 2/5 · log x
E) √5 · log x
√(x⁵) = x^(5/2). Luego log x^(5/2) = (5/2) · log x.
12

A) 2 log a − log b + 3 log c
B) 2 log a + log b − 3 log c

D) (2a + b − 3c) log
E) log 2a + log b + log 3c
Primero separa el cociente, luego el producto. Finalmente baja los exponentes.
13

A) 1,105
B) 1,255
C) 0,778
D) 1,431
E) 0,954
18 = 2 × 3². Aplica log(M · N) = log M + log N y log 3² = 2 log 3.
14
¿Cuál es el valor de log₅(1/125)?
A) −1
B) −2
C) −3
D) 3
E) 1/3
1/125 = 5⁻³. Luego log₅(5⁻³) = −3.

Si log₂(√17 + 1) = y, ¿cuál es el valor de log₂(√17 − 1)?

A) 2 loga 5
B) 0

A) 1/16
B) 1/8

A) 75
B) 75⁻¹
C) 0,03
D) 0,12
E) 0,75
Calcula m, n y x por separado usando la definición. Luego opera.
19
Si x^(log₃ 5) = 25, ¿cuál es el valor de x²?
A) 16
B) 25
C) 9
D) 64
E) 81
Aplica logaritmo en base 3 a ambos lados. Usa la identidad log₃(a^b) = b·log₃ a y log₃ 25 = 2·log₃ 5.
20
¿Cuántas horas (h) deben transcurrir para que un cultivo inicial de N bacterias que se duplica cada hora alcance la cantidad de M bacterias? (La fórmula es T = N · 2h)
A) N + log₂(M)
B) M + log₂(N)
C) log(M + 2N)
D) log₂(N) + log₂(M)
E) log₂(M/N)
Despeja h: M = N · 2^h → M/N = 2^h → h = log₂(M/N).

Si x^(log₃ 5) = 25, ¿cuánto es x²?
1
Aplicar log₃ a ambos lados: log₃[x^(log₃5)] = log₃ 25

Usar log Mⁿ = n · log M: (log₃ 5) · (log₃ x) = log₃ 25

log₃ 25 = log₃ 5² = 2 · log₃ 5
4
Dividir ambos lados por log₃ 5 (≠ 0): log₃ x = 2
5
Por definición: x = 3² = 9
6
Por lo tanto: x² = 9² = 81... pero la alternativa C es 9.
Revisando: la pregunta pide x², pero x = 3, entonces x² = 9. ¡x = 3, no 9!

Corrección: log₃ x = 2 → x = 3². Pero podríamos buscar x directamente:
Si log₃ x = 2, entonces x = 9 → x² = 81.

Con x = 9: 9^(log₃ 5) = (3²)^(log₃ 5) = 3^(2·log₃ 5) = (3^(log₃ 5))² = 5² = 25 ✓
Entonces x = 9, y la pregunta pide x² = 81... si la alternativa correcta realmente es C = 9, entonces la pregunta pedía solo x.

9^(log₃ 5) = (3²)^(log₃ 5) = 3^(2 log₃ 5) = (3^(log₃ 5))² = 5² = 25 ✓
